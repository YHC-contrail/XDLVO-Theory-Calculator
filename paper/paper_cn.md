# 摘要

扩展的Derjaguin-Landau-Verwey-Overbeek（XDLVO）理论是解析膜分离过程中微观界面相互作用、预测膜污染行为的核心物理化学框架[@vanOss1994]。针对当前计算工具普遍存在的对商业软件环境依赖、数据隐私风险以及可视化教学功能缺失等问题，本文开发了XDLVO-Web——一款开源、轻量级且完全基于客户端架构的计算软件。该软件利用标准Web技术（HTML5/JavaScript）实现了在浏览器端直接求解非线性方程组与积分公式，完成了从接触角数据输入到表面能分量、吉布斯自由能及总相互作用能计算的全流程。通过引入物理引擎与Canvas绘图技术，XDLVO-Web独创性地提供了粒子动态交互模拟与实验过程录制功能。作为一款零安装、离线可用且具备交互式可视化特性的工具，它为环境工程与胶体化学领域的研究人员提供了一个高效、安全且直观的分析平台。

# 需求陈述

膜污染是制约膜技术在水处理、生物制药和食品加工等领域高效应用的关键瓶颈[@Shannon2008; @Guo2012]。为了深入理解并控制膜污染，准确解析膜表面与胶体颗粒或微生物之间的微观相互作用力至关重要。经典的DLVO理论[@Derjaguin1941; @Verwey1948]仅考虑范德华力（LW）和双电层静电力（EL），但在亲疏水性显著的膜分离体系中，路易斯酸碱相互作用力（AB）往往起主导作用[@Hermansson1999]。因此，涵盖了AB作用力的扩展XDLVO理论已成为该领域的黄金标准。

然而，XDLVO理论的计算过程繁琐，涉及扩展Young-Dupré方程组的求解及复杂的相互作用能积分，这对科研人员提出了较高的计算门槛。尽管XDLVO理论应用广泛，但目前的计算工具生态系统存在显著的断层。现有的解决方案主要分为两类：

1. **基于商业软件的本地程序**：如基于MATLAB开发的Parti-Suite[@Granda2020]，功能强大但依赖昂贵的商业许可及庞大的运行环境，限制了其在不同操作系统间的迁移与普及。

2. **基于服务器的Web工具**：如Zorzan等人[@Zorzan2025]开发的在线计算器以及Abbott[@Abbott2019]开发的胶体科学App，这类工具虽然无需安装，但计算逻辑依赖远程服务器支持。这种架构不仅要求用户时刻保持网络连接，更迫使通过服务器传输敏感的实验数据，对于涉及未发表数据或工业机密的研究项目存在潜在的隐私泄露风险。

此外，上述工具大多仅能输出静态的数值或能量-距离曲线，缺乏对"能垒"和"力场平衡"等抽象物理概念的直观展示，不利于初学者理解或教学演示。

针对上述局限，XDLVO-Web旨在填补现有工具在轻量化、隐私保护与交互式可视化之间的空白。作为一款完全基于客户端运行的软件，它无需任何后端服务器支持，直接在用户的浏览器中执行所有运算，确保了数据的绝对隐私与零延迟响应。同时，其内置的动态模拟模块为科研验证与学术展示提供了全新的视角。

# 数学模型与技术实现

XDLVO-Web采用纯前端技术栈，严格遵循界面化学热力学原理的核心算法，对表面张力（γ）、吉布斯自由能（ΔG）和相互作用能（U）进行求解。

## 表面张力计算

软件首先依据用户输入的三种探针液体（二碘甲烷、甲酰胺、去离子水）的接触角数据，利用扩展的Young-Dupré方程构建线性方程组：

$$\gamma_l(1+\cos\theta) = 2(\sqrt{\gamma_m^{LW}\gamma_l^{LW}} + \sqrt{\gamma_m^+\gamma_l^-} + \sqrt{\gamma_m^-\gamma_l^+})$$

其中，表面张力的路易斯酸碱分量为：

$$\gamma^{AB} = 2\sqrt{\gamma^+\gamma^-}$$

表面张力总量为：

$$\gamma^{TOT} = \gamma^{LW} + \gamma^{AB}$$

为了在脱离Python或MATLAB后端的情况下实现高精度求解，本软件内置了基于JavaScript实现的线性代数求解模块，通过克莱姆法则与高斯消元法在本地实时解算矩阵。

## 吉布斯自由能计算

在获得表面能参数后，系统依据XDLVO理论计算单位面积的吉布斯自由能：

$$\Delta G^{LW} = -2(\sqrt{\gamma_m^{LW}} - \sqrt{\gamma_w^{LW}})(\sqrt{\gamma_f^{LW}} - \sqrt{\gamma_w^{LW}})$$

$$\Delta G^{AB} = 2\sqrt{\gamma_w^-}[(\sqrt{\gamma_f^-} + \sqrt{\gamma_m^-} - \sqrt{\gamma_w^-}) + (\sqrt{\gamma_f^+} + \sqrt{\gamma_m^+} - \sqrt{\gamma_w^+})] - 2\sqrt{\gamma_f^-\gamma_m^+} - 2\sqrt{\gamma_f^+\gamma_m^-}$$

$$\Delta G^{EL} = \frac{\kappa\varepsilon_0\varepsilon_r}{2}(\zeta_m^2 + \zeta_f^2)[1 - \coth(\kappa h_0) + \frac{2\zeta_m\zeta_f}{\zeta_m^2 + \zeta_f^2}\text{csch}(\kappa h_0)]$$

$$\Delta G^{TOT} = \Delta G^{LW} + \Delta G^{AB} + \Delta G^{EL}$$

其中，h₀为最小分离距离（0.158 nm），εᵣ为水的相对介电常数，ε₀为真空介电常数，ζ表示zeta电位。

## 相互作用能计算

相互作用能作为分离距离的函数：

$$U^{LW}(h) = \frac{2\pi\Delta G^{LW}h_0^2\alpha}{h} \times 10^{-6}$$

$$U^{AB}(h) = 2\pi\alpha\lambda\Delta G^{AB}\exp\left(\frac{h_0-h}{\lambda}\right) \times 10^{-3}$$

$$U^{EL}(h) = \pi\varepsilon_0\varepsilon_r\alpha\left[\zeta_m\zeta_f\ln\left(\frac{1+e^{-\kappa h}}{1-e^{-\kappa h}}\right) + \frac{\zeta_m^2+\zeta_f^2}{2}\ln(1-e^{-2\kappa h})\right] \times 10^{4}$$

$$U^{TOT}(h) = U^{LW}(h) + U^{AB}(h) + U^{EL}(h)$$

其中，α为污染物表观水力半径，λ为极性作用力特征衰减长度（0.6 nm）。

德拜长度的倒数κ根据离子强度和温度动态计算：

$$\kappa = \sqrt{\frac{2N_Ae^2I}{\varepsilon_0\varepsilon_rkT}}$$

水的相对介电常数εᵣ作为温度T的函数：

$$\varepsilon_r = 87.740 - 0.40008(T-273.15) + 9.398\times10^{-4}(T-273.15)^2 - 1.410\times10^{-6}(T-273.15)^3$$

## 可视化与交互

为了实现高效的交互体验与可视化，软件集成了Plotly.js图表库和HTML5 Canvas绘图技术。Canvas模块引入了简化的物理引擎概念，根据当前距离下总作用能的梯度（即作用力）实时计算粒子的加速度与运动轨迹，通过requestAnimationFrame实现流畅的动画渲染。此外，软件集成了MediaRecorder API，支持将动态模拟过程直接录制为WebM格式视频。

# 应用场景与功能特性

XDLVO-Web的设计初衷是服务于广泛的膜技术研究与教学场景。在科研应用中，研究人员可利用其多参数配置面板，快速输入不同改性膜材料的接触角数据及Zeta电位，软件即时响应并重绘能量曲线，辅助快速筛选抗污染性能优异的膜材料。通过调整离子强度和温度参数，用户可以模拟不同水质环境下膜污染趋势的动态变化。

软件提供了完善的数据导出功能，生成的Excel报表包含所有输入参数、中间计算结果（表面张力、吉布斯自由能）以及完整的能量分布数据，精度保留至小数点后6位，满足高精度科研数据后处理的需求。

在教学演示场景中，XDLVO-Web的交互式可视化模块具有独特优势。教师可以通过拖拽滑块改变距离，向学生直观展示"能垒"如何阻挡污染物沉积。配套的视频录制功能允许用户在"时长模式"、"距离模式"或"静止模式"下记录这些动态过程，将抽象的胶体化学原理转化为可视化的多媒体素材。

# 局限性与未来发展

尽管XDLVO-Web在轻量化、隐私保护和交互体验方面具有显著优势，但作为开源项目的初期版本，软件仍存在局限性。我们已将项目源代码托管至GitHub，并诚挚邀请社区开发者共同参与改进：

1. **探针液体数据库扩展**：当前版本已新增"自定义液体"接口，但内置预设库仅包含经典的"三液法"组合。我们期待社区贡献者协助扩充内置液体数据库。

2. **几何模型优化**：当前计算内核基于经典的"板-球"几何假设。前沿研究表明，膜表面粗糙度以及污染物非规则形状对XDLVO相互作用能有显著影响[@Hoek2006]。我们期待具备相关算法背景的开发者协助集成基于表面形貌扫描数据（如AFM数据）的复杂几何校正模型。

3. **数据持久化改进**：鉴于本软件采用"无后端"纯客户端架构，用户的计算配置仅驻留在浏览器临时内存中。我们欢迎社区探索基于IndexedDB或PWA技术的本地存储优化方案。

# 结论

XDLVO-Web是一款集成了严谨数学内核、便捷交互设计与强大可视化能力的分析工具。通过采用纯客户端架构，它成功克服了现有工具对运行环境的依赖及数据隐私的隐患，实现了真正的零门槛、零风险计算。其独创的动态物理模拟功能为膜污染机理的解析提供了新的维度。尽管当前版本在探针液体库和几何模型上仍有拓展空间，但其开源特性为持续优化奠定了基础。XDLVO-Web将依托开源社区的协作，持续为环境工程、材料科学及胶体化学领域的研究人员提供强有力的支持，推动相关理论在实际工程中的应用与发展。
